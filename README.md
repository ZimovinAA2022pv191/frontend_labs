# Introduction
### to start backend, you need:
1) cd online_shop
2) pip install -r requirements.txt
3) python manage.py migrate
4) python manage.py shell
5) from rest_app.controllers.init_db import *
6) InitorSystem().init()
7) exit()

## After init BD:
### python manage.py runserver


# Start ReactApp
### create new terminal
1. cd fronted
2. npm i
3. npm start
